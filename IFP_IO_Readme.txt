Manhunt 1/2 IFP Importer/Exporter

Support Import IFP/BIN Animation
Support binary IFP Ouput
Support Text json file Input/Ouput (For Manhunt Toolkit)
Support binary .anim file Input/Ouput

Q&A:
--------
Q: How to import ifp animation to game?
A: Currently, you need to export the json text file and repack/unpack the IFP / BIN file using the Manhunt Toolkit.  
https://github.com/Sor3nt/manhunt-toolkit
